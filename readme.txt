== Description ==
This Plugin lets you duplicate posts, pages and every other custom post type. Duplicate Posts, You can duplicate your pages, posts and custom post by just one click and it will save as your selected options (draft, private, public, pending).

Cloning works for posts created with Gutenberg or the classic editor as well as with page builders. All the content, the categories, tags, featured image and every setting will be duplicated to a new post that is saved as a draft, private, public, pending etc.

== KEY FEATURES  ==
<ul>
<li><b>User Roles:</b> Allow User Roles To access Duplicate Page.</li>
<li><b>Post Types:</b> Filter to show Duplicate Page link in post types.</li>
<li><b>Duplicate Link Location:</b> Option where to show clone link.</li>
<li><b>Status:</b> Option to select Duplicate Posts Status.</li>
<li><b>Redirection:</b> Option to Redirect after click on clone link.</li>
<li><b>Author:</b> Option to assign author for clone post.</li>
<li><b>Duplicate Link Title:</b> Option to change Duplicate Post Link Title.</li>
<li><b>Post Prefix:</b> Option to add Post Prefix.</li>
<li><b>Post Suffix:</b> Option to add Post Suffix.</li>
</ul>

== HOW TO USE ==
<ol>
<li>First Activate Plugin.</li>
<li>Go Select to <b>Duplicate Page & Post</b> Menu from Settings Tab and savings settings.</li>
<li>Then Create New Post/Page or Use old.</li>
<li>After click on duplicate this link, then duplicate post/ page will be created and saved as draft,publish,pending,private depending upon settings.</li>
</ol>

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

After Plugin Active go to Settings-> Duplicate Page & Post.
